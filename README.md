# skeleton

Build
---

1. ``` cd tools/docker/ ```
2. ``` docker-compose build ```
3. ``` docker-compose up -d ```
4. ``` docker-compose start ```
5. ``` docker-compose exec vanga_web bash ```   
6.1. ``` cd backend/ ```    
6.2. ```  composer install ```  
7.1. ``` cd ../frontend/ ```       
7.2. ``` npm install ```
8. ``` npm start ```
9. Enjoy [http://192.168.0.10:4200/ ](http://192.168.0.10:4200/ )
